# 029-纯虚函数

纯虚函数本质上与Java的抽象方法相同

c++中没有interface关键字，接口也是使用类来定义的

将上一章的虚函数修改为纯虚函数，如下

```C++
class Entity {
public:
    virtual std::string getName() = 0;
};
```
